### 1. Описание проекта
Компания планирует запустить новую акцию, чтобы продать как можно больше туров. Команда разработала два варианта посадочной страницы официального сайта компании и провела A/B-тестирование. 

### 2. Какой кейс решаем
Необходимо проанализировать эффективность обоих вариантов посадочной страницы сразу по двум критериям: конверсии покупки и среднему чеку. На основе проведенного анализа будет необходимо сделать выводы и принять решение: какой из вариантов дизайна более предпочтительный.
**Метрика качества**\
Принято решение о выборе одного из вариантов

**Что практикуем**\
Отрабатываем навыки A/B тестирования.
